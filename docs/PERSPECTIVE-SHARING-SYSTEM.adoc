== The Vision

**"What does YouTube look like for a Black woman? A rural conservative? A teenager in Mumbai?"**

Instead of *hypothetical* personas, users can share their **actual algorithmic experience** and others can temporarily "try it on."

**Goal**: Build empathy by experiencing the internet from others' perspectives

== Core Concept: Algorithmic Experience Profiles

An "Algorithmic Experience Profile" (AEP) is a lightweight data structure that captures:

1. **What the algorithm shows you** (feed composition)
2. **How the algorithm treats you** (recommendations, search results)
3. **What you never see** (filter bubble boundaries)

**NOT captured** (privacy):
- Actual browsing history
- Personal information
- Login credentials
- Anything identifiable

**Captured** (anonymized):
- Feed category distribution (e.g., "70% Gaming, 20% Memes, 10% Commentary")
- Recommendation patterns (e.g., "Algorithm suggests more extreme content over time")
- Bubble boundaries (e.g., "Never see: Politics, News, Education")
- Diversity metrics (e.g., "Simpson's Diversity Index: 0.32")

When user clicks "Try This Perspective":

**Safe Mode (Default)**:
1. Opens 4-5 tabs that represent this perspective's feed
2. **Does NOT engage** (no clicks, no watch time, no likes)
3. User can visually see the difference
4. Close tabs immediately - no profile contamination

**Immersive Mode (Advanced)**:
1. Temporarily modifies your feed to match this profile
2. Runs for 15 minutes (time-boxed)
3. Records what algorithm shows you
4. Reverts to your profile afterwards
5. Generates comparison report

== Privacy & Ethics

**Strict Privacy Rules**:

1. **No PII**: Profiles must strip all personally identifiable information
2. **Opt-in only**: Users must explicitly consent to sharing
3. **Anonymity levels**: Users choose how much to reveal
4. **Right to delete**: Profiles can be revoked from repository
5. **No tracking**: Repository doesn't track who downloads what

**Ethical Consent**:

```json
{
  "ethical_consent": {
    "consents_to_sharing": true,
    "understands_public": true,
    "age_verified": true,
    "can_revoke": true,
    "anonymity_level": "high",
    "signature": "a3f2b1c4...",
    "timestamp": "2026-01-24T12:00:00Z"
  }
}
```

**Anonymization Process**:

```rescript
module ProfileAnonymizer = {
  let anonymizeProfile = (rawFeedState: Observer.FeedState.t): algorithmicProfile => {
    // STRIP personal data
    let anonymized = {
      feedComposition: calculateCategoryDistribution(rawFeedState),
      bubbleBoundaries: identifyBubbleBoundaries(rawFeedState),
      diversityScore: calculateDiversityMetrics(rawFeedState),
      // NO watch history, NO search queries, NO identifiers
    }

    // ADD noise for differential privacy (if requested)
    if anonymityLevel == High {
      addLaplaceNoise(anonymized, epsilon: 0.1)
    } else {
      anonymized
    }
  }
}
```

== Technical Implementation

**File Format**: JSON (human-readable, lightweight)
**File Extension**: `.aep` (Algorithmic Experience Profile)
**File Size**: 5-10KB (tiny!)

**Import/Export API**:

```rescript
module ProfileSharing = {
  // Export current experience as profile
  let exportProfile = (
    feedState: Observer.FeedState.t,
    metadata: {
      description: string,
      anonymityLevel: anonymityLevel,
    }
  ): string => {
    let profile = {
      version: "1.0",
      id: generateUUID(),
      created: Date.now(),
      platform: "youtube",
      metadata: metadata,
      feedComposition: extractFeedComposition(feedState),
      bubbleBoundaries: extractBubbleBoundaries(feedState),
      // ...
    }

    JSON.stringify(profile, null, 2)  // Pretty-printed JSON
  }

  // Import profile from JSON
  let importProfile = (jsonString: string): result<algorithmicProfile, string> => {
    try {
      let parsed = JSON.parse(jsonString)

      // Validate schema
      if !validateProfileSchema(parsed) {
        Error("Invalid profile format")
      } else {
        Ok(parseProfile(parsed))
      }
    } catch {
    | _ => Error("Failed to parse profile")
    }
  }

  // Apply profile (Safe Mode: view only)
  let applyProfileSafeMode = (
    profile: algorithmicProfile
  ): array<string> => {
    // Generate URLs that would match this profile's feed
    let sampleUrls = generateRepresentativeUrls(profile.feedComposition)

    // Open tabs (view only, no engagement)
    sampleUrls->Array.map(url => {
      ChromeTabs.Tabs.createWithUrl(url, ~active=false)
    })

    sampleUrls  // Return for logging
  }
}
```
